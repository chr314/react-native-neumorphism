## Installation

```sh
yarn install react-native-neumorphism
```

## Usage

```js
import Neumorphism from "react-native-neumorphism";

// ...

<Neumorphism
  lightColor={'#dedddd'}
  darkColor={'#979797'}
  shapeType={'flat'}
  radius={50}
>
  <View style={{ width: 200, height: 200}}>
    <Text>TEST</Text>
  </View>
</Neumorphism>
```
